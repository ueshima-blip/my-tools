# my-tools — 授業で使う自作ツール置き場

このフォルダ（リポジトリ）には、授業で使うために作った Web ツール（HTML ファイル）を保存しています。

## 📁 名前のしくみ（GitHub に慣れていない方へ）

GitHub では、持ち物は必ず `アカウント名 / フォルダ名` の形で表示されます。あなたの場合は次のようになっています。

**`ueshima-blip` / `my-tools`**

- **`ueshima-blip`**（前）… あなたの **アカウント名（ログイン ID）** です。すべての持ち物の前に付く「表札」のようなもので、フォルダではありません。
- **`my-tools`**（後ろ）… この **フォルダ（リポジトリ）の名前** です。中に下記のツールが入っています。

## 🧰 入っているツール

| ツール | 説明 | ファイル |
|--------|------|----------|
| エタノールの沸騰実験シミュレーション（理科） | エタノールが沸騰する温度を調べる実験のシミュレーション教材。タッチ操作に対応。 | [`index.html`](./index.html) |
| 提出物チェックシステム | クラスの提出物を管理するツール。Excel から生徒を一括登録できます。 | [`submission-tracker.html`](./submission-tracker.html) |

## ▶️ ツールの開き方

HTML ツールは、GitHub の画面でファイル名をクリックしても「中身のプログラム（コード）」が表示されるだけで、教材としては動きません。実際に使うには、次のどちらかの方法があります。

### 方法1：GitHub Pages をオンにする（おすすめ・設定は一度だけ）

1. このリポジトリの上部メニュー **Settings**（設定）を開く
2. 左側の **Pages** を開く
3. **Branch** を `main` に選んで **Save** を押す

数分後、次のアドレスでブラウザから直接ツールを開けるようになります（リンクを保存しておけば、毎回すぐ使えます）。

- エタノールの沸騰実験シミュレーション：`https://ueshima-blip.github.io/my-tools/`
- 提出物チェックシステム：`https://ueshima-blip.github.io/my-tools/submission-tracker.html`

### 方法2：ファイルをダウンロードして開く

ファイルを開いて右上あたりの **Download**（ダウンロード）ボタンで保存し、保存したファイルをダブルクリックすると、ブラウザで開けます。

## ✏️ 名前を変えたくなったら

- **このフォルダ（リポジトリ）の名前** `my-tools` を変えたい → **Settings** の一番上「Repository name」で変更できます。
- **アカウント名** `ueshima-blip` を変えたい → URL が全部変わる影響の大きい操作です。アカウントの **Settings → Account → Change username** から変更できますが、変える前に一度ご相談ください。
