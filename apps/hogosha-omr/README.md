# 保護者会 調査票読み取りアプリ（OMR）

紙で集めた保護者会の希望調査票（マークシート）をスキャンしたファイル（**PDF / JPG / PNG**）から、塗りつぶしマークを自動で読み取り、Excelツール（[`excel/hogosha-kai/`](../../excel/hogosha-kai/)）に取り込めるCSVを作るブラウザアプリです。

- **インストール不要・全部ブラウザ内で処理**（スキャン画像はどこにも送信されません）
- PDFの表示には同梱の [pdf.js](https://mozilla.github.io/pdf.js/)（Mozilla製・Apache-2.0）を使用

## 使い方

1. Excel側で「**調査票を印刷**」→ 調査票（1人1枚）と `調査票定義.csv` ができる
2. 調査票を **100%（実際のサイズ）** で印刷して配布・回収
3. コピー機などでスキャン（まとめてPDFでも、1枚ずつJPGでもOK。グレースケール・150〜300dpi推奨）
4. このアプリ（`index.html`）を開く
   - GitHub Pages: `https://<ユーザー名>.github.io/my-tools/apps/hogosha-omr/`
   - またはフォルダごとUSB等にコピーして `index.html` をダブルクリックでも動きます
5. ①定義CSV → ②スキャン の順に読み込む
6. ページごとの結果を確認
   - **セルをクリックで ○⇔なし を修正**
   - 黄色の枠 = 自信のない読み取り（必ず目視確認）
   - 出席番号の読み取りに失敗・重複したページは、名簿のプルダウンから選び直す
7. 「CSVを書き出す」→ `希望取込.csv` がダウンロードされる
8. Excel側の「**希望を取り込む**」でそのCSVを選ぶ

## 読み取りのしくみ（メモ）

- 調査票の四隅の黒い四角（マーカー）を検出し、傾き・ズレ・拡大縮小を自動補正
- 左端の黒い縦帯（8ビット+パリティ）から出席番号を自動判定
- マークは「紙の白さ」とくらべた濃さで判定し、際どいものは「要確認」として黄色表示

## 開発者向け: テスト

```bash
cd test
python3 make_synthetic.py   # 擬似スキャン画像（回転・ノイズ・薄い鉛筆）を生成
node test_omr.js            # 読み取りエンジンの自動テスト
```

`omr.js` は DOM 非依存の純粋ロジックで、Node.js でそのままテストできます。
