# タスクボード

仕事中にデスクへ常駐させる、コンパクトな **タスク管理ウィジェット**です。**上に「本日のタスク」・下に「今週中のタスク」**を縦に並べて表示します。タスクに「締め切り・作業する日・作業する週」を設定しておくと、**その日・その週が来たら自動でボードに表示**されます（作業日が今日になると自動で「本日」へ繰り上がります）。

一般的な Todo アプリと差別化するため、**3つのデザイン（NEON / GLASS / MONO）を切り替え**できます（選択は保存されます）。

> Windows での利用を想定。インストール不要・完全オフライン・外部送信なし。データはこの端末のブラウザ内（localStorage）にだけ保存されます。

## 主な機能

- **縦型2段・コンパクト**：上＝本日／下＝今週。デスクの端に置けるサイズ
- **アプリ内でタスク登録**：「＋ タスクを追加」から
  - **締め切り**（任意の日付）… 〆チップで表示。今日／超過は赤く強調
  - **作業する日**（任意）… その日が来たら自動で「本日」に表示（`今日`／`明日`ボタンつき）
  - **作業する週**（今週／来週／指定なし）… その週のあいだ「今週」に表示
  - **優先**（先頭に表示）
- **自動振り分け**：設定に応じて本日／今週へ自動表示。来週以降のものは「予定 N」とだけ表示して、その週が来たら自動で出てきます
- **クイック追加**：各段の下の入力にサッと書いて `Enter`（末尾 `!` で優先）
- **完了で消す**：丸をクリック → 下の「完了」へ。`完了を消す` でまとめて削除（**元に戻す**つき）
- **編集**：タスクの ✎（またはダブルクリック） ／ **優先**：☆ ／ **削除**：✕
- **3デザイン切替**：右上の ◐ ボタンで NEON → GLASS → MONO（選択は保存）
- **目隠し**：右上の ◑ ボタン、または `Esc` キーで中身を時計表示に（授業前にサッと隠す）
- **バックアップ**：右上 `⋯` メニューから `.json` 書き出し／読み込み

## 開き方（ワンクリック）

`タスクボード起動.bat` を**ダブルクリック**。Edge または Chrome が**枠なしの小窓（アプリ表示）**で開きます。データ保存用の専用プロファイルを使うので、ふだんのブラウザには影響しません。

> 💡 `タスクボード起動.bat` を右クリック →「ショートカットの作成」でデスクトップに置くと、アイコンから一発起動できます。

`index.html` を**ダブルクリック**しても、既定のブラウザで開いて使えます。

## 🪟 デスクトップにウィジェットのように常駐させる

ブラウザのウィンドウを、付箋のようにデスクトップへ浮かせて常駐させる手順です。**方法A が一番かんたんで、オフラインでも動きます。**

> ⚠️ **「アプリとしてインストール」がグレーアウト／見当たらないとき**
> この機能は **`https`（または `localhost`）で開いたときだけ**有効です。`index.html` を直接ひらく／起動bat（=`file://`）では選べません。アプリとして入れたい場合は **方法B（GitHub Pages）** を使ってください。

### 方法A：起動bat ＋ PowerToys（インストール不要・オフラインOK・おすすめ）

1. `タスクボード起動.bat` をダブルクリック → Edge/Chrome が**枠なしの小窓**で開きます（タブも住所バーも無し）。
2. **常に最前面に固定（＝ウィジェット化）**：Windows標準には最前面固定が無いので、Microsoft 公式の無料ツール **PowerToys** を使います。
   - Microsoft Store で「**PowerToys**」を検索 → 入手
   - タスクボードの窓を選んだ状態で **`Win`+`Ctrl`+`T`** → 他の窓の上に**浮いたまま固定**（もう一度で解除）
3. 窓の端をつかんで小さくし、画面の右端などに置けば付箋ウィジェットの完成。
4. ワンクリック起動にしたいときは、`タスクボード起動.bat` を右クリック →「ショートカットの作成」でデスクトップに置く。

> 上級者向け：最前面固定の自動化は **AutoHotkey** のワンライナー（`WinSet, AlwaysOnTop` 系）でも可能です。

### 方法B：アプリとしてインストール（アイコン化したい場合・要 https）

「アプリとしてインストール」を使うには、`https` で開く必要があります。このリポジトリなら **GitHub Pages** が手軽です。

1. リポジトリの **Settings → Pages** で Branch を `main` にして保存（設定は一度だけ）
2. 数分後、Edge/Chrome で **`https://ueshima-blip.github.io/my-tools/apps/task-board/`** を開く
3. 右上の **「…」メニュー** から「**アプリとしてインストール**」（Edge）／「**ページをアプリとしてインストール**」「**ショートカットを作成 → ウィンドウとして開く**」（Chrome・版により名称が違います）。住所バーに出る「インストール（⊕や画面アイコン）」からでもOK
4. タブ・住所バーの無い**専用ウィンドウ＋独自アイコン**になり、スタート/タスクバーに固定できます（次回からワンクリック）
5. 仕上げに 方法A の **PowerToys `Win`+`Ctrl`+`T`** で最前面に固定

> このアプリは Service Worker を内蔵しているため、一度 https で開いて読み込めば、その後は**オフラインでも**起動できます。

### 授業前にサッと隠す

- **中身だけ隠す**：右上 ◑ ボタン、または `Esc` キー（時計表示に。もう一度で戻る）
- **ウィンドウごと隠す**：`Win`+`D`（デスクトップ表示）が最速。`Win`+`D` でまた戻せます

## 保存について

- タスクはこの端末のブラウザ内（localStorage）に保存されます。**同じ場所の `index.html`**（または同じ起動方法）を使い続ければ、次に開いても残っています。
- フォルダを別の場所へ移すと保存先が変わり、前のタスクが見えなくなることがあります。引っ越し前に `⋯` から **バックアップを書き出し**、移動先で**読み込み**てください。
- **起動方法は統一を**：`タスクボード起動.bat`（専用プロファイル）と `index.html` のダブルクリック（ふだんのブラウザ）、「アプリとしてインストール」では、それぞれ保存場所が別になります。どれか一つに決めて使うと混乱しません（おすすめは「アプリとしてインストール」または bat）。
