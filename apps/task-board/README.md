# タスクボード

仕事中に使う、デスクに常駐させる **タスク管理ツール**です。「本日のタスク」と「今週中のタスク」を並べて表示し、終わったらチェックして消します。一般的な Todo アプリと差別化した、**3つのデザイン（NEON / GLASS / MONO）を切り替えられる**のが特長です。

> Windows での利用を想定。インストール不要・完全オフライン・外部送信なし。データはこの端末のブラウザ内（localStorage）にだけ保存されます。

## 主な機能

- **2つのリスト**：本日のタスク／今週中のタスク を左右に表示
- **完了で消す**：チェックすると下の「完了」へ。`完了を消す` でまとめて削除（**元に戻す**つき）
- **クイック入力**：
  - 締切タグ … 末尾に `@金` や `@15:00` → 締切チップ
  - 優先 … 末尾に `!`、または ☆ ボタン
  - 例）`中間テスト作問 @金 !`
- **編集**：文字を**ダブルクリック**
- **移動**：⇄ で「本日 ↔ 今週」を入れ替え
- **並べ替え**：ドラッグで上下入れ替え
- **3デザイン切替**：右上の `NEON / GLASS / MONO`（選んだデザインは保存されます）
- **進捗バー**：完了率を下部に表示
- **目隠し（授業前にサッと隠す）**：右上の `Esc 目隠し` ボタン、または `Esc` キーで中身を時計表示に。もう一度で戻ります
- **バックアップ**：`⤓` から内容を `.json` で書き出し／読み込み（端末の引っ越し・保険に）

## 開き方

### いちばん簡単（ワンクリック）

`タスクボード起動.bat` を**ダブルクリック**。Edge または Chrome が**枠なしのアプリ表示**で開きます（データ保存用の専用プロファイルを使うので、ふだんのブラウザには影響しません）。

> 💡 よく使うなら、`タスクボード起動.bat` を右クリック →「ショートカットの作成」でデスクトップに置くと、アイコンから一発で起動できます。

### もっと手軽に

`index.html` を**ダブルクリック**しても、既定のブラウザで開いて使えます。

### スマホ／別PCからも（任意）

リポジトリで GitHub Pages を有効にしている場合は、ブラウザから直接開けます。
`https://ueshima-blip.github.io/my-tools/apps/task-board/`

## 授業で端末を持っていくときに隠す

- **中身だけ隠す**：`Esc 目隠し` ボタン（または `Esc` キー）。画面が時計表示になり、もう一度で戻ります。
- **ウィンドウごと隠す**：`Win`+`D`（デスクトップ表示）が最速。`Win`+`D` でまた戻せます。

## 保存について

- タスクはこの端末のブラウザ内（localStorage）に保存されます。**同じ場所の `index.html`** を使い続ければ、次に開いても残っています。
- フォルダを別の場所へ移すと、別の保存先になり前のタスクが見えなくなることがあります。引っ越し前に `⤓` から **バックアップを書き出し**、移動先で**読み込み**てください。
- **起動方法は統一を**：`タスクボード起動.bat`（専用プロファイル）と `index.html` のダブルクリック（ふだんのブラウザ）では保存場所が別になります。どちらか一方に決めて使うと混乱しません（おすすめは bat）。
