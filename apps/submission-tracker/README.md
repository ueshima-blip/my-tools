# 提出物チェックシステム

クラスの提出物を管理するツールです。Excel から生徒を一括登録でき、複数クラスに対応しています。

## 主な機能

- **生徒の一括登録**：Excel／CSV の「年」「組」列からクラスを自動判別し、クラスごとに名簿を登録（個別入力・テキスト貼り付けにも対応）
- **複数クラス対応**：画面上部のクラス切替バーで、追加・切替・名前変更・削除
- **固定メニュー**：上部のメニュー（タブ・クラス切替）はスクロールしても常に表示
- **提出状況チェック**：◯提出／✕未提出／△未記入をワンクリックで切替
- **集計**：生徒別の提出率を一覧表示。表示する提出物を選べます（**提出期限前／提出期限後／締切から1週間以内／個別選択**）
- **印刷**：
  - チェック一覧（選んだ提出物の名簿）を **A4 縦** で印刷
  - 集計表（集計ページで選択中の複数提出物）を **A4 横** で印刷
  - **未提出のお知らせ**を **B5 縦** で印刷（✕未提出・△未記入がある生徒へ、その子の未提出一覧を1人1枚）
- **データ保護**：データはブラウザ内（localStorage）にのみ保存（外部送信なし）。バックアップの書き出し／読み込みに対応

## 使い方

`index.html` をブラウザで開くだけで動きます（インストール不要）。入力したデータはブラウザ内（localStorage）に保存されます。
