# 席替えアプリ

教員向けの席替え支援アプリ。クラス名簿（Excel）を取り込み、男女配置・班分け・座席固定・距離条件などの制約を考慮した上で、瞬時／ルーレットで席替えを実行できる単一HTMLファイル製のローカルアプリです。

## 主な機能

- **クラス × 教室ごとの独立した座席設定** — 同じ「第１理科室」でも1組と2組で別々のレイアウトを保持
- **教室レイアウト編集** — 列・行数指定、座席の有効／無効／使わない席、男子席／女子席指定、班色割り当て
- **距離条件・座席固定** — 「2人を離す」「特定生徒は前から3行以内」等の条件指定、特定席への固定
- **席替え方式が3つ**
  - 瞬時に決定
  - 一人ずつルーレット
  - 一括ルーレット
- **ルーレット中断時の自動再開** — ブラウザがクラッシュしても続きから再開可能
- **印刷／JPG保存** — A4横向き1ページ自動収まり、月日のみを右下に表示
- **印刷デザインを選べる** — 「シンプル（標準）／水彩教室／ポップ／シック」から選択。黒板ヘッダー・花飾り・教壇ボックスなどの装飾を付けつつ、男女色・班色はそのまま保持。生徒視点・教師視点（180°回転）どちらでも正立表示。画面に即プレビューされ、印刷ダイアログでも切り替え可能
- **履歴** — 過去の席替え結果をクラス×教室ごとに最大20件保持・復元可能
- **バックアップ（外部保存）** — 全データ（名簿・教室・座席設定・班・条件・固定・履歴・デザイン設定）を1つのJSONにまとめて外部保存。iPad/対応端末では「📲 共有／送信」でiCloud Drive・Google Drive・メール等へワンタップ送信、PCでは「📤 ファイルに書き出す」。しばらく保存していないと設定タブに催促（●印）を表示。復元（インポート）も対応
- **データ消去メニュー** — 履歴のみ／名簿のみ／教室のみ／全データ完全消去

## 使い方

1. `index.html` を Chrome／Safari／Edge などで開く
2. 名簿タブで Excel（.xlsx）を読み込む（列：出席番号・氏名・ふりがな・性別）
3. 配置タブで教室の列・行を設定し、座席を編集
4. 必要なら条件タブで距離条件・座席固定を追加
5. 上部の「⚡ 瞬時に決定」または「🎰 一人ずつルーレット」「🎲 一括ルーレット」で席替え
6. 印刷前に上部「🎨 デザイン」で見た目を選び、「🖨 印刷」「💾 JPG保存」で出力
7. 定期的に「設定」タブからバックアップを外部（クラウド・メール・USB等）へ保存

## データ保存場所

すべてブラウザの localStorage に保存されます。サーバーへの送信は一切ありません。

そのため、ブラウザのデータ消去・端末の故障・買い替えなどで消える可能性があります。**万が一に備え、「設定」タブの「データのバックアップ（外部保存）」から定期的に外部へ保存してください。**

- **📲 バックアップを共有／送信**（iPad など対応端末）… iCloud Drive・Google Drive・メールなどへワンタップで送れます
- **📤 ファイルに書き出す**（PC など）… JSON ファイルをダウンロードし、クラウドや USB に保存
- **📥 バックアップから復元**… 書き出した JSON を読み込んで元に戻します（端末の引っ越しにも使えます）
- しばらくバックアップしていないと、「設定」タブに催促の●印と注意が表示されます

## 動作環境

- 単一HTMLファイル。インストール・サーバー不要
- iPad（Safari）でも動作確認済み
- Excel 読み込み用に [SheetJS](https://sheetjs.com/) を CDN から、JPG保存用に html2canvas を CDN から読み込みます
