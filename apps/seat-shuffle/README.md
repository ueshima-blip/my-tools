# 席替えアプリ

教員向けの席替え支援アプリ。クラス名簿（Excel）を取り込み、男女配置・班分け・座席固定・距離条件などの制約を考慮した上で、瞬時／ルーレットで席替えを実行できる単一HTMLファイル製のローカルアプリです。

> 📄 **操作マニュアル（PDF）**：[`席替えアプリ_操作マニュアル.pdf`](席替えアプリ_操作マニュアル.pdf)（教員向け・全機能の使い方）。PDFは `python3 build_manual.py`（`reportlab` 必須）で再生成できます。

## 主な機能

- **名簿テンプレート（Excel）** — 名簿タブの「名簿テンプレート（Excel）をダウンロード」で、列（出席番号・氏名・ふりがな・性別）入りの空のひな形を取得 → 記入して読み込むだけ。配布用のひな形は [`名簿テンプレート.xlsx`](名簿テンプレート.xlsx)（`python3 build_roster_template.py` で再生成可）
- **クラス × 教室ごとの独立した座席設定** — 同じ「第１理科室」でも1組と2組で別々のレイアウトを保持
- **教室レイアウト編集** — 列・行数指定、座席の有効／無効／使わない席、男子席／女子席指定、班色割り当て
- **距離条件・座席固定** — 「2人を離す」「特定生徒は前から3行以内」等の条件指定、特定席への固定
- **席替え方式が3つ**
  - 瞬時に決定
  - 一人ずつルーレット
  - 一括ルーレット
- **ルーレットは自動で全画面表示** — 「一人ずつ／一括ルーレット」を始めると自動でブラウザを全画面化。教室のモニター・プロジェクターに大きく映せます。終了時は「✓ 全画面を解除して終了」ボタンでワンクリック解除（上部の「⛶ 全画面」ボタンでも切替可）
- **ルーレット中断時の自動再開** — ブラウザがクラッシュしても続きから再開可能
- **印刷／JPG保存** — A4横向き1ページ自動収まり、月日のみを右下に表示
- **印刷デザインを選べる** — 中学・高校でも使える落ち着いたデザインを用意。「シンプル（標準）／机（モダン）／机（木目）／シック」から選択。
  - **机（モダン）／机（木目）** … 普通教室向け。座席ひとつひとつが学校机の形（天板・前面の縁・脚）になり、前面に黒板／ホワイトボードと教卓を配置
  - 男女色・班色は机の天板色としてそのまま保持。生徒視点・教師視点（180°回転）どちらでも正立表示。画面に即プレビューされ、印刷ダイアログでも切り替え可能
- **履歴** — 過去の席替え結果をクラス×教室ごとに最大20件保持・復元可能
- **バックアップ（外部保存）** — 全データ（名簿・教室・座席設定・班・条件・固定・履歴・デザイン設定）を1つのJSONにまとめて外部保存。iPad/対応端末では「📲 共有／送信」でiCloud Drive・Google Drive・メール等へワンタップ送信、PCでは「📤 ファイルに書き出す」。しばらく保存していないと設定タブに催促（●印）を表示。復元（インポート）も対応
- **データ消去メニュー** — 履歴のみ／名簿のみ／教室のみ／全データ完全消去

## 使い方

1. `席替えアプリ_オフライン版.html`（配付用）を Chrome／Safari／Edge などで開く
2. 名簿タブで Excel（.xlsx）を読み込む（列：出席番号・氏名・ふりがな・性別／「名簿テンプレート（Excel）をダウンロード」で空のひな形を取得できます）
3. 配置タブで教室の列・行を設定し、座席を編集
4. 必要なら条件タブで距離条件・座席固定を追加
5. 上部の「⚡ 瞬時に決定」または「🎰 一人ずつルーレット」「🎲 一括ルーレット」で席替え
6. 印刷前に上部「🎨 デザイン」で見た目を選び、「🖨 印刷」「💾 JPG保存」で出力
7. 定期的に「設定」タブからバックアップを外部（クラウド・メール・USB等）へ保存

## データ保存場所

すべてブラウザの localStorage に保存されます。サーバーへの送信は一切ありません。

そのため、ブラウザのデータ消去・端末の故障・買い替えなどで消える可能性があります。**万が一に備え、「設定」タブの「データのバックアップ（外部保存）」から定期的に外部へ保存してください。**

- **📲 バックアップを共有／送信**（iPad など対応端末）… iCloud Drive・Google Drive・メールなどへワンタップで送れます
- **📤 ファイルに書き出す**（PC など）… JSON ファイルをダウンロードし、クラウドや USB に保存
- **📥 バックアップから復元**… 書き出した JSON を読み込んで元に戻します（端末の引っ越しにも使えます）
- しばらくバックアップしていないと、「設定」タブに催促の●印と注意が表示されます

## 動作環境

- 単一HTMLファイル。インストール・サーバー不要
- iPad（Safari）でも動作確認済み

## 2つの版（配付向け）

用途に合わせて2つのファイルを用意しています。**人に配るときは、完全オフラインの `席替えアプリ_オフライン版.html` がおすすめです。**

| ファイル | 特徴 | 向いている用途 |
|---|---|---|
| `席替えアプリ_オンライン版.html` | 軽量（約210KB）。フォント・Excel処理・画像保存の部品を CDN（[SheetJS](https://sheetjs.com/)／html2canvas）から読み込みます | GitHub Pages での公開・開発（編集の原本） |
| `席替えアプリ_オフライン版.html` | **完全オフライン版**。上記の部品をすべて内蔵し、**インターネット接続が一切不要**（約1.3MB） | USB・共有フォルダ・メールでの配付 |

> **プライバシー**：どちらの版も、名簿や生徒名を**外部に送信することはありません**（処理はすべて端末内で完結）。オンライン版がネットから取得するのは「プログラム部品」だけで、生徒情報は含まれません。オフライン版は外部接続そのものが無いため、ネットの無い教室でもそのまま使えます。

`席替えアプリ_オフライン版.html` は、`席替えアプリ_オンライン版.html` を元に `python3 build_offline.py` で再生成できます（部品を内蔵した配付用ファイルを書き出します）。
