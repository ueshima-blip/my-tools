# かんじノート — iPad 学習用ノートアプリ

子ども向けの学習用ノートアプリです。Goodnotes など標準的なノートアプリの機能（ペン／マーカー／消しゴム／色／太さ／ページ／Undo/Redo）に加え、**書いた漢字が正しいかを自動で判定し、まちがっていたら正しい字を教える機能**を備えています。Apple Pencil ／その他のスタイラスペンに対応しています。

## 使い方

1. このフォルダの `index.html` を Safari（iPad）で開きます。
2. ホーム画面に追加すると全画面アプリのように使えます（`apple-mobile-web-app-capable`）。
3. インストールやサーバーは不要。HTML / CSS / JS だけで動作します。

開発中は次のいずれかでローカルに配信できます：

```sh
# Python があれば
python3 -m http.server 8000

# Node.js があれば
npx http-server -p 8000
```

…そして `http://(マシンのIP):8000/` を iPad の Safari で開きます。

## 画面と操作

### モード

- **📝 ノート**: 罫線入りのページに自由に書きます。Goodnotes と似た感覚です。
- **🈯 漢字練習**: お題の単語を入れると、文字数ぶんのマス（原稿用紙風）が出ます。マスの中に書くと、書き終わるたびに自動で正誤を判定します。

### 道具

- **ペン**：筆圧で線の太さが変わります（Apple Pencil 推奨）。
- **マーカー**：半透明の太い線。重ねると濃くなります。
- **けしゴム**：触れたストロークを丸ごと消します（Goodnotes と同じ「ストローク消し」）。

### 色／太さ

色パレットから 6 色、太さスライダーで 1〜14。

### Pencil のみ（パームリジェクション）

「Pencilのみ」を ON にすると、`pointerType === 'touch'` のイベント（手のひら／指）を完全に無視し、Apple Pencil（および他のスタイラスペンが pen として認識される場合）の入力だけを受け付けます。これにより、書いている途中に手のひらが画面に当たっても誤って線が引かれません。

### ページ

ノートモードでは複数ページに対応。`◀ ▶` で前後移動、`＋` で追加。

### 保存／読込

`💾 保存` を押すとブラウザの `localStorage` に保存されます。`📂 読込` で復元。

### キーボードショートカット（PC 開発時）

- `Ctrl/Cmd + Z` … Undo
- `Ctrl/Cmd + Shift + Z` または `Ctrl/Cmd + Y` … Redo
- `1` ペン / `2` マーカー / `3` 消しゴム

## 漢字練習モードと「まちがい検出」

1. **「お題」**に練習したい単語を入力（例: `学校`、`犬と猫`、`花が咲く`）。
2. 文字数ぶんのマスが画面に並びます（マスにはお手本がうすく表示）。
3. それぞれのマスに書きます。書き終わると（700ms 静止後）自動で判定。
4. 判定結果：
   - **○（緑）**: 正解。
   - **×（赤）**: ちがう字。マスの下に「正しくは『◯』」と表示し、サイドパネルで「これは『△』に近いよ」と、子どもが書いてしまった字を伝えます。
   - **？（橙）**: 書きかけ／判別不能。「もう少していねいに書いてみよう」。

「自動はんてい」をオフにすれば「いま判定する」ボタンで一括判定します。

### 認識の仕組み（`recognizer.js`）

オフラインの軽量テンプレートマッチで実装しています：

1. ストロークから外接矩形を取って 64×64 の画像に正規化（マージン 10%）。
2. 候補となる漢字（お題＋まぎらわしいグループ＋常用 1・2 年生）をシステム搭載の日本語フォント（Hiragino Mincho 等）で同じ 64×64 に描画。各候補は初回のみキャッシュ。
3. 両方に 3×3 のボックスフィルタをかけ、コサイン類似度を計算。
4. 最大スコアの字が「お題」と一致する → ○。明確に別の字が最大スコア → ×（その字を「これに近い」として提示）。

「まぎらわしいグループ」は、子どもが特に混同しやすい字（例：`日／月／目／白／百／自`、`木／本／末／未／体`、`人／入／八／大／犬／太／天／夫`、`土／士／王／玉／主`、`大／犬／太` 等）を強制的に候補に含めます。これにより、目標字とよく似た誤字を見つけやすくなります。

### 認識の限界

- ストロークの順序や数は使っていません（純粋な見た目の類似度）。たとえば「日」と「目」は形が似ているので、書きかたによっては判別を誤ることがあります。サイドパネルの「くわしい結果」で上位候補が確認できます。
- 候補プールに無い珍しい漢字は、よく似た字に誤判定される場合があります。お題に入れた字は必ず候補になるため、正しく書けば ○ が出ます。

## 構成ファイル

```
index.html       … 画面構成
style.css        … iPad 向けの大きめタッチ領域 / 罫線 / マス目スタイル
app.js           … ノート本体（描画、ツール、ページ、保存、UI）
recognizer.js    … 漢字認識（テンプレートマッチ）
```

依存ライブラリは無し。

## 動作環境

- iPad（Safari 14+）+ Apple Pencil、またはスタイラスペン
- 動作確認用に PC ブラウザ（Chrome / Edge / Safari）でも開けます（マウスで描画可能）
