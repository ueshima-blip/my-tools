# 保護者会調整ツール ver2.0（Excelマクロ）

保護者会（個人面談）の希望日程を集めて、**ボタン一つで全員分の時間割を自動調整**するExcelツールです。

ネットで公開されているフリーソフト「エクセル 保護者会希望調整マクロ ver1.11」（作者: Yorono Sake さん、2015年公開）をベースに、大幅に機能追加・改良したものです。オリジナルの添付文書は [`オリジナル添付_readme_v111.txt`](オリジナル添付_readme_v111.txt)（Shift-JIS）、無改造のオリジナルは [`保護者会調整_ver111_オリジナル.xlsm`](保護者会調整_ver111_オリジナル.xlsm) として保存してあります。

## ver2.0 の新機能・改良点

### 1. 調整エンジンを刷新 — 「調整できません」がなくなりました
- 旧版は運まかせの試行錯誤方式（×50回試して失敗、など）でしたが、**数学的に確実な方法（増加路法）** に置き換えました。組み合わせが存在すれば**必ず一回で**見つかります。
- どうしても組めない場合は、**原因の生徒グループを特定して表示**します（「番号 3, 15, 28 の3名の希望が2コマに集中しています」のように）。誰に希望を増やしてもらえばよいかが一目で分かります。
- 調整方法・試行回数を選ぶ画面は不要になったため廃止しました。

### 2. △（第2希望）が調整に使われるようになりました
- 旧版の△はただのメモでしたが、ver2.0では「**○だけで組めないときに限り**△も使う」2段階調整になりました。△で確定した枠も青色表示されます。

### 3. 保護者向け「個別通知票」の自動作成
- 予定表作成のときに「通知票も作成しますか?」と聞かれます。**一人ずつの切り取り式お知らせ**（氏名・日時入り、A4に10名分）が自動で出来上がります。手書き転記が不要になります。

### 4. 希望の取り込み（手入力からの解放）
Start! シートの「**希望を取り込む**」ボタンで、次のファイルから希望を自動入力できます:
- **Googleフォームの回答**（CSVダウンロード）
- **Microsoft Forms の回答**（Excelダウンロード）
- **OMR読み取りアプリのCSV**（紙の調査票をスキャンして読み取ったもの・下記）

紙で集める場合の道具も2つ:
- 「**調査票を印刷**」ボタン → マークシート式の調査票（1人1枚・名前と機械読み取り用マーク入り）を自動作成。回収後にスキャン（PDF/JPG）して、ブラウザアプリ [`apps/hogosha-omr/`](../../apps/hogosha-omr/) で読み取るとCSVになります。
- 「**クイック入力**」ボタン → 紙を見ながら「1-3 1-5 2-10」のようにキーボードで高速入力。

### 5. 最新のExcelへの対応・安定化
- 古い ActiveX ボタンを廃止し、新しいOfficeでも無効化されない図形ボタンに変更。
- 全マクロにエラー処理を追加（途中で止まって画面が固まったままになる事故を防止）。
- 確定結果を色とは別に内部にも記録し、誤って色を変えてしまっても予定表作成時に検出・修復できます。
- 旧版のバグ（休憩解除が無関係な行まで消す、日順指定の入力でエラー落ち等）を修正。

## 使い方（基本の流れ）

1. **名簿** シートに、出席番号(2桁)・氏名(1セル)の2列を貼り付け（最大50名）
2. **Start!** シートの「初期設定」→ 日数(2〜5)・面談時間・開始/最終時刻を設定
3. **希望入力** シートの灰色セルに日付（例: `6/20`）を入力
4. 希望を入力（取り込み / クイック入力 / 手入力のどれでも）
   - 休憩したい時間帯は「メニュー」→「希望入力・休憩設定」で設定
5. 「メニュー」→ **調整作業** → 自動で全員確定
6. 「メニュー」→ **作成** → 予定表 + 個別通知票（+ PDF保存も選べます）
7. 微調整したいときは、調整シートで「メニュー」→ 調整作業 →「はい」（入れ替えモード）

## フォームで希望を集める場合の作り方

GoogleフォームやMicrosoft Formsで集めるときは、次のようにすると自動取り込みできます:

- **出席番号**を聞く質問を入れる（質問名に「番号」を含める）※氏名でも照合できますが番号が確実
- 時間帯の選択肢は **`6/20 15:30`（月/日 半角スペース 時:分）** の形式にする
  - チェックボックス（複数選択）1問でも、時間帯ごとのはい/いいえでもOK
  - 「第2希望」の質問を分ける場合は、質問名に「第2希望」と入れると△として取り込まれます

## 紙（スキャン）で集める場合

1. Start! の「調査票を印刷」→ 1人1枚の調査票と `調査票定義.csv` ができる
2. **必ず100%（実際のサイズ）で印刷**（「ページに合わせる」禁止）
3. 配布・回収 → コピー機でまとめてスキャン（PDFかJPG、グレースケール推奨）
4. ブラウザで OMR読み取りアプリ を開く → 定義CSVとスキャンを読み込む → 結果を確認(修正)してCSV書き出し
5. Excelの「希望を取り込む」でそのCSVを選ぶ

## 動作環境・注意

- Windows + Excel 2013以降（Microsoft 365 推奨）。**マクロを有効にして**使ってください。
- 初めて使う前に、ダミー名簿で一度通し（名簿→設定→希望→調整→予定表）を試すことをおすすめします。
- ファイルは OneDrive 上ではなくローカル（デスクトップ等）に置くとPDF保存・定義ファイル出力が確実です。

### ⚠️ 最初に必ず：「マクロの実行がブロックされました」と赤く表示されたら

GitHubなどネットからダウンロードしたマクロ付きファイルは、Microsoftの仕様（2022年〜）で必ず一度ブロックされます。故障ではありません。次の手順で解除します（ダウンロードしたファイルごとに1回だけ）:

1. Excelをいったん閉じる
2. ダウンロードした `保護者会調整_ver200.xlsm` を右クリック →「プロパティ」
3. 「全般」タブ最下部の「セキュリティ: このファイルは他のコンピューターから取得したものです…」の **「許可する」にチェック** → OK
4. ファイルを開き直し、黄色いバーの「コンテンツの有効化」をクリック

※OneDriveやネットワークドライブ上だと「許可する」が表示されないことがあります。先にデスクトップ等へ移動してください。
※毎回の解除が面倒な場合は、Excelの「ファイル → オプション → トラストセンター → トラストセンターの設定 → 信頼できる場所」に保存フォルダを登録すると不要になります。
