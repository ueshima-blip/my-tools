# 自動ルビ振り（ふりがな）— かんたん配布パッケージ

Word の文章の漢字に、**ふりがな（ルビ）を自動で振る**マクロです。
元になっているのは「初心者備忘録さん」「福原裕之の科学グッズ」さんが公開されている Word マクロで、Word 内蔵の「ふりがなガイド」を自動で実行する仕組みです（Windows 版 Word 専用）。

このフォルダは、そのマクロを **「テキストをコピーして VBA に貼り付ける」作業なしで、他の先生に配れる形**にまとめたものです。

---

## 😣 これまでの配り方の問題

配られた先生は、毎回こんな作業が必要でした。

1. Word を開く → 表示 → マクロ → 名前を入力 → 作成
2. VBA という黒っぽい画面が開く
3. **もとからある文字を全部消す**
4. **長いプログラムをコピーして貼り付ける**
5. 保存する

→ 3・4 のあたりで「どこを消すの？」「貼る場所が違う」と**つまずく先生が続出**。これが広まらない一番の原因でした。

## 🎯 新しい配り方（コピペ作業ゼロ）

**マクロを“入れ込み済み”の Word ファイルを1個作って、それを配るだけ**にします。
受け取った先生は **VBA 画面を一度も開きません**。

配り方は2通り。おすすめは方法Aです。

| | 方法A：**インストール版**（イチオシ） | 方法B：**そのまま版**（インストール不要） |
|---|---|---|
| 配るもの | `AutoRuby.dotm` ＋ `install.bat` | `自動ルビ.docm` 1個 |
| 受け取る先生の操作 | `install.bat` をダブルクリック → Word 再起動 | ファイルを開いて、その中で作業 |
| 使える範囲 | **これ以降ひらく全部の Word 文書** | その1ファイルの中だけ |
| マクロの警告 | **出ない**（安全な場所に入るため） | 開くとき「コンテンツの有効化」を1回押す |
| 向いている人 | パソコンに置いて長く使う先生 | とりあえず1回試したい先生 |

> 💡 どちらの方法でも、受け取る先生は**プログラムを一切さわりません**。配る側が下の「作る手順」で1回だけ準備します。

---

## 👩‍🏫 受け取る先生むけ：使い方（配布物にこの3行を添えるだけ）

### 方法A（インストール版）を受け取ったら
1. `install.bat` を **ダブルクリック**（「実行」をきかれたら はい／実行）
2. 開いている **Word をいったん全部閉じて、もう一度開く**
3. リボンに出てくる **「ルビ振り」タブ**のボタン、または **表示 → マクロ** から実行
   - 元に戻したいときは `uninstall.bat` をダブルクリック

### 方法B（そのまま版）を受け取ったら
1. `自動ルビ.docm` を **ダブルクリックで開く**
2. 上に出る黄色い帯の **「コンテンツの有効化」を1回押す**
3. その文書に文章を打つ／貼り付けて、**「ルビ振り」タブ**または **表示 → マクロ** から実行

### 4つのボタンの意味（共通）
| ボタン | はたらき |
|---|---|
| **全文にルビ** | 文書全体の漢字すべてにルビ |
| **全文にルビ（初出のみ）** | 同じ漢字は最初の1回だけルビ（プリント向き） |
| **選択範囲にルビ** | マウスで選んだ範囲の漢字すべてにルビ |
| **選択範囲にルビ（初出のみ）** | 選んだ範囲で、同じ漢字は最初の1回だけ |

## 🛠 配る側むけ：配布ファイルを作る手順（準備は1回だけ）

すでにマクロが動く Word ファイルをお持ちなら、**5分**で配布物が作れます。

### STEP 1. マクロを“きれいに”入れ直す（コピペより簡単・任意）
このフォルダの [`macro/AutoRuby.bas`](macro/AutoRuby.bas) は、**VBA 画面にドラッグするだけ**で取り込めます（コピペ不要）。

1. Word で 表示 → マクロ → 適当な名前で「作成」→ VBA 画面を開く
2. 左の「プロジェクト」欄に、`AutoRuby.bas` を **ドラッグ＆ドロップ**
   （または メニューの ファイル → ファイルのインポート → `AutoRuby.bas` を選ぶ）
3. すでに古いモジュールがあれば、右クリック → 解放（削除）しておく

> もとのマクロをそのまま使っている場合は、この STEP 1 は飛ばして STEP 2 へ進んでも大丈夫です。

### STEP 2-A. 方法A用：テンプレート（.dotm）として保存
1. ファイル → 名前を付けて保存
2. ファイルの種類で **「Word マクロ有効テンプレート (*.dotm)」** を選ぶ
3. ファイル名を **`AutoRuby`** にして保存（→ `AutoRuby.dotm` ができる）
4. できた `AutoRuby.dotm` を、このフォルダの [`installer/`](installer/) の中の
   `install.bat` ／ `uninstall.bat` ／ `open_startup_folder.bat` と**同じフォルダにまとめ**、
   フォルダごと ZIP にして配る

### STEP 2-B. 方法B用：そのまま使う文書（.docm）として保存
1. ファイル → 名前を付けて保存
2. ファイルの種類で **「Word マクロ有効文書 (*.docm)」** を選ぶ
3. ファイル名を **`自動ルビ`** にして保存 → この1ファイルを配るだけ

### STEP 3.（任意）リボンに「ルビ振り」ボタンを出す
ボタンがあると、表示→マクロを開かずワンクリックで使えます。無料ツールで1回設定すれば、配布物に引き継がれます。

1. 無料の **「Office RibbonX Editor」** を入手して起動
   （「Office RibbonX Editor」で検索／GitHub の fernandreu/office-ribbonx-editor で配布）
2. STEP 2 で作った `AutoRuby.dotm`（または `.docm`）を開く
3. メニュー Insert → **Office 2010+ Custom UI Part** を選ぶ
4. 追加された `customUI14.xml` の中身を、このフォルダの
   [`ribbon/customUI14.xml`](ribbon/customUI14.xml) の中身に**置きかえて保存**
5. Word で開き直すと、リボンに **「ルビ振り」タブ**が出ます

> ボタンを使わない場合は STEP 3 を飛ばしてOK。受け取る先生は **表示 → マクロ** から選んで実行できます。
> もっと手軽にボタンを出すなら、各先生が **ファイル → オプション → クイックアクセスツールバー → 「マクロ」から4つを追加** する方法もあります（XML 不要）。

## 📦 このフォルダの中身

```
word/auto-ruby/
├── README.md                     ← この説明書
├── macro/
│   └── AutoRuby.bas              マクロ本体（VBAにドラッグで取り込める／コピペ不要）
├── ribbon/
│   └── customUI14.xml            リボンに「ルビ振り」タブを出す定義（任意）
└── installer/
    ├── install.bat               AutoRuby.dotm をWordのSTARTUPに入れる
    ├── uninstall.bat             元に戻す
    └── open_startup_folder.bat   STARTUPフォルダを手動で開く（保険）
```

> ⚠️ `AutoRuby.dotm` / `自動ルビ.docm` は **配る人が STEP 2 で作るファイル**です。
> 中身に IME 由来の読み・マクロが含まれるため、この公開リポジトリには置いていません。

## ❓ 困ったとき

- **Mac で動かない** … このマクロは **Windows 版 Word 専用**です（内蔵ふりがな機能を使うため）。
- **`install.bat` が「実行できない」「ブロックされた」** … 学校の PC でバッチが止められている場合は、
  `open_startup_folder.bat` でフォルダを開き、`AutoRuby.dotm` を**手でドラッグして入れて**ください。
  それも難しければ、方法B（`.docm` をそのまま開く）を案内すれば確実です。
- **「ルビ振り」タブが出ない** … 方法Aは **Word の再起動**が必要です。再起動しても出ないときは
  表示 → マクロ から実行できるか確認してください（タブが無くてもマクロは使えます）。
- **読みが違う漢字がある** … Word／IME の自動判定の読みです。気になる箇所は、その漢字を選んで
  ホーム → ルビ（亜のアイコン）から手で直せます。
