# Excel ファイル置き場

進度予定表など、Excel で作った教材・ツールをここに入れます。

## 入れ方の目安

- 1 つのファイルだけのものは、この `excel/` フォルダに直接置いてかまいません
  （例: `進度予定表_2026年度.xlsx`）。
- 複数のファイルや説明資料がセットになっているものは、専用のサブフォルダを作って
  まとめます（例: `excel/席替え抽選/`）。

## 名前の付け方（おすすめ）

- 日本語のファイル名でも大丈夫です。
- 何のファイルか分かる名前 ＋ 年度や日付を付けると、後で探しやすくなります。
  - 例: `進度予定表_2026年度.xlsx`、`成績集計_2026-04.xlsx`
